# StarHR Malaysian Statutory Compliance Guide

This documents the statutory deduction rules implemented in StarHR for Malaysian payroll.

---

## EPF (KWSP) - Employees Provident Fund

### Current Rates (2024)

| Employee Age | Employee Rate      | Employer Rate                   |
| ------------ | ------------------ | ------------------------------- |
| Below 60     | 11%                | 12% (≤RM5,000) / 13% (>RM5,000) |
| 60 and above | 0% (optional 5.5%) | 4%                              |

### Calculation

```
Employee Contribution = Basic Salary × Employee Rate
Employer Contribution = Basic Salary × Employer Rate
```

**References:**

- [KWSP Official Website](https://www.kwsp.gov.my/)
- [EPF Contribution Table](https://www.kwsp.gov.my/employer/contribution-downloads)

## SOCSO (PERKESO) - Social Security

### Contribution Table (2024)

| Monthly Wages        | Employee | Employer |
| -------------------- | -------- | -------- |
| ≤ RM30               | 0.10     | 0.40     |
| RM30.01 - RM50       | 0.15     | 0.60     |
| RM50.01 - RM70       | 0.25     | 0.85     |
| ...                  | ...      | ...      |
| RM4,900.01 - RM5,000 | 24.75    | 86.65    |
| > RM5,000            | Capped   | Capped   |

### Categories

1. **Employment Injury Scheme** - All employees
2. **Invalidity Scheme** - Employees below 60

**References:**

- [PERKESO Official Website](https://www.perkeso.gov.my/)

## PCB (MTD) - Monthly Tax Deduction

### Tax Brackets (2024)

| Chargeable Income       | Rate |
| ----------------------- | ---- |
| First RM5,000           | 0%   |
| RM5,001 - RM20,000      | 1%   |
| RM20,001 - RM35,000     | 3%   |
| RM35,001 - RM50,000     | 6%   |
| RM50,001 - RM70,000     | 11%  |
| RM70,001 - RM100,000    | 19%  |
| RM100,001 - RM400,000   | 25%  |
| RM400,001 - RM600,000   | 26%  |
| RM600,001 - RM2,000,000 | 28%  |
| > RM2,000,000           | 30%  |

**References:**

## Implementation in StarHR

### Location

Statutory calculations are in:

- `api/lib/statutory-rules.ts` - Rate tables and logic
- `api/routes/payroll.ts` - Payroll processing

### Updating Rates

When statutory rates change:

1. Update rate tables in `statutory-rules.ts`
2. Add effective date
3. Run tests: `npm run test:api`
4. Document in this file
